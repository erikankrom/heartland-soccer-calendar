### Task 2: Annotate event rows with scores and opponent record

In the event row rendering logic (where each game in `events` is rendered as an HTML row):

**Past games (game date < today AND matched game exists in `results.games`):**
- Find matching result by normalizing the event's DTSTART date to `YYYY-MM-DD` using `icalDateToISO` logic (replicate in client JS as a simple inline function)
- Append to the row: a score badge like `W 3–1` or `L 0–2` or `T 1–1` in a styled `<span>`
- Color coding: green for W, red for L, gray for T

**Upcoming games (game date >= today AND opponent found in `opponentRecords`):**
- Find the opponent ID: parse the event's SUMMARY to extract the opponent team number (same leading-digits pattern as server side)
- Look up `opponentRecords[opponentId]`
- If found, append to the row: `Opp: 2W–3L–0T` in a muted `<span>`
- If not found, render nothing extra

**Date matching approach for past games:**
```js
function dtToISO(dtstart) {
  const raw = dtstart.includes(':') ? dtstart.split(':').pop() : dtstart;
  return raw.slice(0,4) + '-' + raw.slice(4,6) + '-' + raw.slice(6,8);
}
```

Build a lookup map from `results.games` keyed by `date` string for O(1) matching:
```js
const resultsByDate = {};
for (const g of results.games) {
  if (g.scored) resultsByDate[g.date] = g;
}
```
